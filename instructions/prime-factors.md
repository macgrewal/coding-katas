# Coding Kata - Prime Factors

An attempt at completing the Prime Factors problem using TDD.

## Information
This project uses:
- Scala 2.11.8
- SBT 0.13.8
- ScalaTest 2.2.6

##Requirements
Give a number return a list of prime factors when multiplied result in the number passed in

## Compile instructions
```
sbt clean compile
```

## Test instructions
```
sbt clean test
```
